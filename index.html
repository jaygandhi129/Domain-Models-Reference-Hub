Getting Started

How to Use This Wiki

This reference guide provides detailed information about various industry-standard domain models. Each section includes:

  • Overview and purpose of the domain model
  • Core entities and their relationships
  • Key concepts and terminology
  • Implementation guidelines and best practices
  • Links to official specifications and resources

Navigation

Use the sidebar to navigate between different domain models. Each model is organized into logical sections for easy reference.

Adding New Models

To add a new domain model, simply update the data.js file with your model information. The structure includes:

{
  id: "unique-id",
  name: "Model Name",
  description: "Brief description",
  category: "Category Name",
  tags: ["tag1", "tag2"],
  entities: [...],
  concepts: [...],
  links: [...]
}
